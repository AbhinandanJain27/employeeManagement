The empManagement.sql File can be used to retrieved Data and add it to your database but before doing so 

Create a database named - employee_management in your mysql

Run this command after creating the database

mysql -u username -p < empManagement.sql


Before Running this project don't forget to give password of your mysql user in the applications properties file.

If you don't want the previous database and want to create a new one just don't use the .sql dump file just change your username and password for mysql and you are all set.
